# NoUpdateMCBE

## Описание
Самописная программа py которая отключает авто-обновление MCBE GDK+APPX

## Режимы работы
- **Первый режим**: останавливает службу которая связанная с обновлением игры GDK+APPX
- **Второй режим**: обходит запуск лаунчера и проверки обновление для GDK (+быстрый запуск и мультиокна)
